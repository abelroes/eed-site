# Projeto Site EED v3 — Escritório de Design e Arquitetura

## O que é
Site institucional de um escritório de design e arquitetura.

## Formato do site
- **One-pager**: tudo em uma única página HTML
- **Scroll por seções**: cada seção ocupa 100% da tela (viewport)
- **Sem scroll contínuo**: o scroll "pula" de seção em seção (scroll-snap)
- Funciona como âncoras — cada seção é uma "página" independente visualmente

## Páginas (seções)
1. **Home** — tela de impacto, sem header/footer, apenas imagem centralizada
2. **Produtos** — header + imagem central com laterais (carrossel) + footer
3. **Serviços** — layout idêntico ao de Produtos
4. **Sobre** — texto à esquerda + imagens à direita + imagem inferior esquerda
5. **Contato** — imagem grande à esquerda + imagem menor à direita + footer com 3 textos

## Wireframes
Localizados em `Wireframe/`. Cores indicam:
| Cor | Elemento |

| **Magenta** | Viewport (limites da tela) |

> Os alinhamentos nos wireframes são **propositais** e devem ser respeitados.

## Tecnologia
- **HTML + CSS + JS** puros (sem frameworks)
- Responsivo
- Wireframes atuais são para **desktop** (foco inicial)

## Fases futuras
- Design e implementação da versão **mobile**
- Versões em **4 idiomas**: Português BR (principal), Inglês, Espanhol, Chinês
